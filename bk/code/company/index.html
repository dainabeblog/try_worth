Greeting

代表挨拶

芹澤　慶行

トライワースは
誰もが「挑戦」できる
環境を提供する会社です

「挑戦」を通して一人一人の人生を華やかなものにすることを目標に掲げて会社を設立しました。 最初はパソコン1台から動画を作り、仲間が集まり、動画にマッチしたドローンを始めて紆余曲折ありながらも現在に至ります。 最近は、水中ドローンという日本であまり知られていない業界に参入し、地盤を固め、皆さんが手軽に水中ドローンを楽しめるような時代を作ろうと「挑戦」しています。 また皆様に挑戦できる環境を提供するとともにトライワース自身も挑戦していき社会の発展を目指していきたいと思います。

Corporate philosophy

企業理念

「一人一人の人生に革命を」

企業理念とはトライワースの企業としての目的・存在意義を表すものです。 一人一人が何かに「挑戦」をして、それがどんな形であれ挑戦してよかったな。と思える瞬間を提供し、「人生に革命を起こす」という思いが込められています。 いつの時代も世界を変えた人間は何かに「挑戦」してきました。 これからも皆様と「挑戦」を通じて共に社会の発展を目指します。
